## 📋 版本概述

v0.5 Alpha版本是云南省企业就业失业数据采集系统的第一个正式发布版本，包含了完整的用户认证、权限管理、企业备案审批、系统管理和审计留痕等核心功能。

## 🚀 技术架构

### 技术栈
- **后端框架**: FastAPI 0.135.2
- **ASGI服务器**: Uvicorn 0.42.0
- **数据验证**: Pydantic 2.11.9
- **身份认证**: JWT (python-jose)
- **密码加密**: Passlib + PBKDF2_SHA256
- **前端可视化**: Streamlit 1.55.0

### 项目结构
```
SRS/
├── main.py                    # FastAPI应用入口
├── dashboard.py              # Streamlit可视化大屏
├── api/                      # API路由模块
│   └── routers/
│       ├── auth.py          # 身份认证路由
│       ├── province.py      # 省级业务路由
│       ├── admin.py         # 系统管理路由
│       └── audit.py         # 审计日志路由
├── core/                     # 核心功能模块
│   ├── dependencies.py      # 依赖注入和权限检查
│   ├── security.py          # 安全认证
│   ├── swagger_ui.py       # 自定义Swagger界面
│   └── audit.py            # 审计留痕和数据保护
├── db/                       # 数据库模块
│   └── mock_db.py          # 模拟数据库
└── schemas/                  # 数据模型
    └── api_models.py       # API数据模型
```

## 📊 API接口统计

### 接口总数: 22个

#### 身份认证中心 (3个)
- `POST /api/v1/auth/login` - 用户登录
- `GET /api/v1/auth/me` - 获取当前用户信息

#### 省级管理业务流 (1个)
- `POST /api/v1/province/audit-filing` - 企业备案审批

#### 系统管理中心 (11个)
- `GET /api/v1/admin/survey-periods` - 获取调查期列表
- `GET /api/v1/admin/survey-periods/{period_id}` - 获取调查期详情
- `POST /api/v1/admin/survey-periods` - 创建调查期
- `PUT /api/v1/admin/survey-periods/{period_id}` - 更新调查期
- `DELETE /api/v1/admin/survey-periods/{period_id}` - 删除调查期
- `GET /api/v1/admin/users` - 获取用户列表
- `GET /api/v1/admin/users/{user_id}` - 获取用户详情
- `POST /api/v1/admin/users` - 创建用户
- `PUT /api/v1/admin/users/{user_id}` - 更新用户
- `DELETE /api/v1/admin/users/{user_id}` - 删除用户
- `GET /api/v1/admin/audit-logs` - 获取操作审计日志

#### 审计日志查询 (7个)
- `GET /api/v1/audit/logs` - 查询审计日志
- `GET /api/v1/audit/logs/{log_id}` - 获取审计日志详情
- `GET /api/v1/audit/logs/enterprise/{enterprise_id}` - 查询企业操作日志
- `GET /api/v1/audit/logs/user/{user_id}` - 查询用户操作日志
- `GET /api/v1/audit/statistics` - 获取审计统计信息
- `GET /api/v1/audit/data-protection/rules` - 获取数据保护规则
- `POST /api/v1/audit/data-protection/validate` - 验证数据变更合法性

## 🔐 安全特性

### 身份认证
- JWT令牌认证，有效期120分钟
- 密码PBKDF2_SHA256加密存储
- 用户状态管理（激活/禁用）

### 权限控制
- 基于角色的访问控制(RBAC)
- 接口级权限检查
- 越权操作自动拦截

### 审计留痕
- 所有操作自动记录
- 完整的审计信息（用户、时间、操作类型、前后值、原因等）
- 多维度查询和统计分析

### 数据保护
- 字段级保护（关键字段不可修改）
- 记录级保护（重要记录不可删除）
- 操作级保护（敏感操作需要特殊权限）
- 数据完整性检查
