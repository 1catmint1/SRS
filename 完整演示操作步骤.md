## 🎯 完整演示流程

### 第一部分：企业用户功能演示

#### 1.1 企业用户登录
**操作步骤**：
1. 打开浏览器访问 http://127.0.0.1:8000/docs
2. 找到 `1. 身份认证中心` → `POST /api/v1/auth/login`
3. 点击 "Try it out"
4. 填写登录信息：
   ```
   username: enterprise
   password: password123
   ```
5. 点击 "Execute"
6. 复制返回的 `access_token`

**预期结果**：
- 返回状态码 200
- 获得 access_token
- 显示用户信息（企业用户）

---

#### 1.2 企业信息修改（备案信息填报）
**操作步骤**：
1. 找到 `4. 企业管理模块` → `POST /api/v1/enterprise/filing`
2. 点击 "Try it out"
3. 在 Authorization 字段填入：`Bearer {你的token}`
4. 填写企业备案信息：
```json
{
  "enterprise_id": 1001,
  "enterprise_name": "昆明市某制造企业",
  "organization_code": "91530100MA6K1ABC12",
  "enterprise_nature": "国有企业",
  "industry": "制造业",
  "main_business": "机械制造、设备生产",
  "contact_person": "张三",
  "contact_address": "云南省昆明市五华区",
  "postal_code": "650000",
  "contact_phone": "13800138000",
  "fax": "0871-12345678",
  "contact_email": "contact@example.com"
}
```
5. 点击 "Execute"

**预期结果**：
- 返回状态码 200
- 显示备案信息提交成功
- 备案状态为"待审核"
